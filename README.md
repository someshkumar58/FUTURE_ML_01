# Sales Forecasting using Machine Learning

## Project Overview
This project predicts future sales using historical Walmart data.

## Techniques Used
- Data Cleaning
- Feature Engineering
- Random Forest Model

## Results
- MAE and RMSE used for evaluation
- Forecast for next 30 days generated

## Insights
- Sales show seasonal patterns
- Holidays impact demand
- Useful for business planning

## Tools
- Python
- Pandas
- Matplotlib
- Scikit-learn
